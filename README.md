# Employee Salary & Workforce Insights Dashboard (Power BI)

This project analyzes employee salary trends, hiring patterns, and workforce distribution using Power BI.

## Features
- Salary trends by year, month, quarter, and day  
- Employee-level comparisons  
- Hiring patterns for 2023 vs 2024  
- Clear visuals using bar charts, line charts, and pie charts  

## Tools Used
- Power BI  
- Excel
